This is a fluid grid designed to fit cleanly inside all screen sizes. At mobile sizes, the width is always 100% in order to maximize screen real estate. At sizes above 640px wide, the width is 90% of the viewport.

The columns and gutters change width based on the size of the window/device, so no targeting for devices is necessary. If needed, certain column widths can be locked to fit ads, for example. Likewise, the entire site can lock at a certain width within a certain range to allow for pushdown ads & brandscaping.

Designing for a flexible grid involves choosing certain benchmark resolutions, and at those particular sizes all images appear at 100% size.

The layout changes at 480, 768, 1024 and 1600px
