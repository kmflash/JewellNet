Front End Library Page Structure

Please format any additions to the Front End Library as follows.

<section class="fel content"> <!-- The wrapper for all page content [REQUIRED] -->
    <p class="fel intro"> ... </p>

    <!-- BEGIN ITEM GROUP -->
    <section class="fel" id="grid">
        <h2 class="fel">Grid</h2>

        <!-- BEGIN ITEM -->
        <h4 class="fel" id="item-name"> ... </h4>
        <p class="fel [intro]"> ... </p>                                  <!-- [.intro] : makes text larger -->
        <div class="fel-component">
                                                                          <!-- Add [.full|.half|.third|.quarter] to set width of elements inside .fel-component -->
            <div class="fel-example">                                     <!-- [.fel-left] : attach to left side if 2 cols -->

                *** PUT ALL GK MARKUP HERE ***

            </div>
            <pre class="prettyprint [linenums] [lang-*] [fel half] ">     <!-- [linenums] : show line numbers -->
                                                                          <!-- [lang-*] : syntax highlighting for (html|css|less|js) -->
                    *** PUT ALL CODE EXAMPLES IN HERE ***
                    ***      WHITESPACE MATTERS       ***

                    You MUST escape open/close brackets with
                    &lt; and &gt; respectively.

            </pre>
        </div>
        <!-- END ITEM -->

        <!-- [Repeat as needed] -->

    </section>
    <!-- END ITEM GROUP -->
</section>
<!-- END CONTENT -->

FEL Menu Structure

This is the structure of the navigation menu, which appears in a burger nav on xs->sm and as a left rail on md. It goes inside div.fel.menu -> div.fel.menu-inner at the bottom of the DOM. 

 <ul class="fel"> <!-- Top -->
    <li class="fel"><a href="some_page.html" class="fel">Top Level FEL Page</a></li>
    <li class="fel"><a href="this_page.html" class="fel">This Page</a></li>

    <!-- BEGIN menu for this page -->
    <ul class="fel sub" id="pageMenu"> <!-- This is the container for all the menu items on this page -->

        <li class="fel"><a href="#mySection" class="fel menu-page-section">A section</a> <!-- A group -->

            <ul class="fel sub menu-sub-section"> <!-- Children of that section -->

                <li class="fel"><a href="#mySection-item" class="fel">An Item</a></li>
                <li class="fel"><a href="#mySection-item2" class="fel">Another Item</a></li>

            </ul> <!-- END section children -->

        </li> <!-- END section -->

        <!-- [repeat as needed] -->

    </ul> <!-- END #pageMenu -->

</ul> <!-- END top menu -->

Helper Classes

There are a number of helper classes available to format the Front End Library.

Where to Use Class Notes
.fel-component .noflex Turn off flex positioning
Inside .fel-component .full Makes width of element 100% of the content well
.half Makes width of element 50% of the content well
.third Makes width of element 33.3% of the content well
.quarter Makes width of element 25% of the content well
<p> .intro Makes text size larger
Any element .pad Adds 20px padding all around
.fel-content .grey-bg Adds light grey background

Vendors

While most of Genius Kitchen, including the CMS that powers it is custom-build by our team, we do use 3rd parties for certain elements of the geniuskitchen.com website.

Cloudinary

Practically all images are stored & served from cloudinary AWS servers. Using their service allows for on-the-fly image transformations hosted around the world on Amazon CDN servers. Documentation can be found here.

  • Website: cloudinary.com/

Filestack (formerly filepicker.io)

Before Cloudinary, Filestack (formerly filepicker.io) hosted UGC images. Since the adoption of Cloudinary, all uploads now go through that process. But many legacy images remain on the filestack servers.

  • Website: www.filestack.com/

Gigya

Gigya is used for user authentication on Genius Kitchen. GK uses styled "screensets" for log in/sign up modals.

  • Website: www.gigya.com

Helix Business Solutions

Helix hosts, maintains & operates the GK customer service pages.

  • Website: helix.solutions

Perfect Sense Digital

PSD hosts & maintains Recipe Box for Genius Kitchen as well as Food Network.

  • Website: www.perfectsensedigital.com
