Editorial Style Guide

Heroes

Many curated pages on Genus Kitchen contain large photos, or hero images.

These images have logos, menus and other UI elements sitting above, so care should be taken to ensure all text is readable and nothing clashes, visually. You can download a PDF version of these guidelines here.

General Guidelines

Name Height Width
Package Hero 1025 × 376
Home Hero 1025 × 437
WOW Home Hero 1025 × 576

*Image resolution should be 300 ppi

  • Clean, precisely staged set ups look far more fun and organic when placed in layout.
  • Colorful aerial shots with saturated base colors (i.e. non white background) work best.
  • Leave “room for copy” and/or breathing room between materials.
  • Shoot wide to allow for GK's unique cropping needs.
  • Things to avoid: busyness, congestion, stark white or washed out imagery, lots of different colors close together under copy areas.

Watch on Web Hero

Please note, the Watch on Web home hero is cropped on the XS breakpoint, so extra care should be given to the lower 3rd, as well as sides of the hero image to ensure nothing important is hidden.

Character Counts

While most all user-generated components of GK have hard-coded character limits to prevent breaking, shifting or otherwise obscuring other page elements, most editorially-programmed fields do not have such restrictions.

This is to allow greater flexibitlity when programming content. There are, however "best practice" limits determined by Design that should allow for optimal programming. Unless there is a specific need to use more characters, please avoid going over these limits.

All character counts include spaces.
